05 — Usage Rules
Do
  • Use the wordmark on dark backgrounds (#080808, #0F0F0F, #161616) as primary placement
  • Use gold (#C8972A) exclusively for accent — never as a primary background at scale
  • Let Cormorant Garamond carry emotional weight — it earns the space it takes
  • Use DM Mono for all labels, tags, buttons, and metadata — never for body copy
  • Give the mark room — clear space equal to the height of the tallest stroke on all sides
  • Use the mark alone for profile photos, favicons, and watermarks
  • Pair italic Cormorant with the gold color for headline emphasis only
Don't
  • Don't place the logo on busy photographic backgrounds without a dark overlay
  • Don't use gold text on white or light backgrounds — it lacks contrast
  • Don't stretch, distort, rotate, or rearrange the lockup elements
  • Don't use DM Sans for display headlines — it is a body and UI font only
  • Don't introduce additional typefaces — the system is complete at three
  • Don't use bright colors, gradients, or drop shadows on any brand element
  • Don't use the full wordmark at sizes below 120px wide — use the mark only
06 — Voice & Tone Reference
Direct

No hedging. No corporate softening. The audience has been through real things — they can handle real language. Say the hard thing plainly.

Earned

Authority comes from experience, not credentials. The framework exists because it was lived. Every claim is backed by the story behind it.

Constructive

The orientation is always forward. Not recovery, not therapy — architecture. The language of building, not healing. Structure, sequence, foundation.
